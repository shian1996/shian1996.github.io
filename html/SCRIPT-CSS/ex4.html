    固定定位 相對定位以及絕對定位

    1. < div style ="width: 20px; height: 20px; background-color: blue;" >< /div>
    < div style ="width: 120px; height: 120px; background-color: red;" >< /div>
    " #沒有任何設定 =對照組


    2. < div style =" position: fixed; right: 20px; width: 20px; height: 20px; background-color: blue;" >< /div>
    < div style ="width: 120px; height: 120px; background-color: red;" >< /div>
    " #position: fixed 固定定位 滑動網頁 藍色方框always固定在一個位置 也可以給top bottom right left讓他固定在網頁上的指定位置

    3. < div style =" position: relative; top: 20px; width: 20px; height: 20px; background-color: blue;" >< /div>
    < div style ="width: 120px; height: 120px; background-color: red;" >< /div>
    " #position: relative相對定位 藍色方框相對於原本位置做偏移 如果是top是代表原本位置在我的上方 所以我會下移

    階層上的定位

    1. < div style ="padding: 30px; border: 5px solid red;" >
    < div style ="padding: 30px; border: 5px solid red;" >
    < div style ="width: 30px; height: 30px; background-color: blue;" >
    < /div>
    < /div>
    < /div>
    " #沒有任何設定 =對照組

    2. < div style ="padding: 30px; border: 5px solid red;" >
    < div style ="padding: 30px; border: 5px solid red;" >
    < div style =" position: absolute; right: 20px; height: 30px; background-color: blue;" >
    < /div>
    < /div>
    < /div>
    " #position: absolute:絕對定位 使用方法跟原理都固定定位一樣 只是不會隨著網頁移動 在這就是定位在與網頁的右方距離20px的位置上

    3. < div style ="padding: 30px; border: 5px solid red;" >
    < div style =" position: relative; padding: 30px; border: 5px solid red;" >
    < div style =" position: absolute; right: 20px; height: 30px; background-color: blue;" >
    < /div>
    < /div>
    < /div>
    " #絕對定位的話 如果外層也有做定位 那絕對定位就會定位在外層的定位上 在這就是定位在與外層的右方距離20px的位置上 如果有很多外層都有定位的話 那就依照最近的外層來做定位
