    #link 外部css file要在head中間做
    1. < head>
    < link rel ="stylesheet" href ="./style.css"
    < /head>

    style.css 的內容如下 (如果看懂後可以參考ex6來熟悉)
    .food{ color: red; }
    .people{ color: black; }
    .mjt{ background-color: green; }
    .mzt{ background-color: gray; }
    .kmt{ background-color: blue; }
    #在.css file中給予class的設定用.加上標籤名稱,並在大括號中填入style的變數 #也就是說在body中有用到class="標籤名",就可以共用同個style設定
    #css-h1{ background-color: gray; color: yellow; }
    #在.css file中給予id的設定用#加上標籤名稱,並在大括號中填入style的變數 #跟class不同,不可多個標籤共用
    2. < p id ="css-h1" > hello < /p>

    hello

    #在這邊p標籤吃了id的css-h1標籤因此style為背景灰色字體黃色
    3. < p class ="food" > hello < /p>

    hello

    #在這邊p標籤吃了class的food標籤因此style為字體紅色
    4. < p class ="kmt" > hello < /p>

    hello

    #在這邊p標籤吃了class的kmt標籤因此style為背景藍色
    5. < p > hello < /p>

    hello

    #在這邊p標籤沒有被分類 因此為預設
