CSS padding &margin


Code

    1. < a style ="color: rgb(2, 2, 2); background-color: aqua;">今天冷鋒南下 好冷 < /a>
    " #style="background-color"為將a標籤背景塗色,如果是p標籤為整行都塗色

    今天冷鋒南下 好冷

    2. < a style =" padding: 10px; color: rgb(0, 0, 0); background-color: aqua;">今天冷鋒南下 好冷 < /a>
    " #padding = 文字方框內部空間的寬度" 填一個數字等於四邊都空多少*px

    今天冷鋒南下 好冷

    3. < a style =" padding: 10px 20px 3px 4px; color: rgb(0, 0, 0); background-color: aqua;">今天冷鋒南下 好冷 < /a>
    " #四個的話 依序為top(10px)、right(20px)、bot(3px)、left(4px)

    今天冷鋒南下 好冷

    4. < a style =" padding: 10px; margin: 10px; color: rgb(0, 0, 0); background-color: aqua;">今天冷鋒南下 好冷 < /a>
    " #margin = 文字方框外部左右側空間的寬度 也可直接指定左右margin-left or margin-right"

    今天冷鋒南下 好冷
